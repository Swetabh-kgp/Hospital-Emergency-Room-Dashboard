# Hospital Emergency Room Dashboard - Excel Project
## Project Overview
This project is an Excel-based Hospital Emergency Room Dashboard created to analyze emergency room performance and patient flow.
The dashboard helps track important hospital metrics such as patient count, average wait time, satisfaction score, admission status, department referrals, patient attendance status, gender-wise analysis, and age group distribution.
## Dashboard Features
- Total number of patients
- Average wait time
- Patient satisfaction score
- Admission status analysis
- Department referral analysis
- Patient attendance status
- Gender-wise patient analysis
- Age group-wise patient analysis
- Monthly filter buttons
- Year selection option
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Formatting
- Data Visualization
## Purpose of the Dashboard
The purpose of this dashboard is to convert hospital emergency room data into clear and useful insights. It helps understand patient flow, waiting time, admission trends, department referrals, and overall ER performance.

- `Hospital_Emergency_Room_Dashboard.png` - Dashboard preview image
